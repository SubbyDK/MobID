I have started this crazy project to collect all IDs for every mob in the game.

I believe it would be incredibly valuable to have a centralized place where other addons can easily get the **Mob ID** for any creature. This database will make creating certain community addons for Turtle WoW much easier.

The file you need to send is located here:
`\Turtle WoW\WTF\Account\[YOUR ACCOUNT NAME]\SavedVariables\MobID.lua`

The addon collects the **mob name**, **mob level**, and the **zone** you are currently in.

I then use this information to find the corresponding Mob ID on the official Turtle WoW database, and I manually enter it into this project's master database.
